大驼峰命名风格

主要适用于类名、接口名、枚举类名、record类名，默认是排除以下字符之后对命名进行大驼峰风格匹配: AO|BO|DAO|DO|DTO|PO|VO

可以配置类名白名单，和配置覆盖关键字白名单。

其中类名白名单以应为逗号或分号分隔多个类名，关键字白名单则采用覆盖的方式替换原来的白名单关键字列表，原来的关键字列表为: (AO|BO|DAO|DO|DTO|PO|VO), 如再增加 MQ关键字，则配置内容为: (AO|BO|DAO|DO|DTO|PO|VO|MQ)

反例

    // ABC(或AB,BC)不在白名单内
    public class StudentABC {
    }

    // 首字母应大写
    public class studentAO {
    }

正例

    // DTO是白名单内的字符串
    public class StudentDTO {
    }
    // VO是白名单内的字符串
    public class StudentVO {
    }
