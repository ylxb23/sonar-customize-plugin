大驼峰命名风格

主要用于类名、枚举类名、接口名、record类型的命名检查，排除以下字符之后对命名进行大驼峰风格匹配：AO|BO|DAO|DO|DTO|PO|VO

反例

ManTOWomanService.java ->
    // TO(或 OW,TOW)不是白名单中的关键字，故命中大驼峰命名规则
    public class ManTOWomanService {
    }

正例

PersonDO.java ->
    public class PersonDO {
    }
PersonPO.java 文件：
    public class PersonPO {
    }
